Project Overview

Tech Nest is a feature-rich, native Android e-commerce application tailored specifically for selling tech products like laptops, computer accessories, and hardware components. It was developed as an individual assignment to demonstrate proficiency in Java and mobile UI/UX principles.

The app focuses on providing a clean, modern interface where users can easily browse products, search for specific items, and experience a seamless digital shopping journey.

Key Features

  • Dynamic Home Screen with promotional banners
  • Categorized product browsing (Laptops, Peripherals, Components)
  • Detailed product view with dynamic pricing and specifications
  • Shopping cart functionality and checkout flow
  • Clean, modern UI/UX customized with XML layouts and drawables

Technical Stack

Built entirely using native Android development tools, emphasizing code structure, layout design, and smooth user interactions.

Java Android Studio XML Layouts Material Design Drawables & Assets
